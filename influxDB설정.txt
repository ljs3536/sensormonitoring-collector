터미널(또는 명령 프롬프트)을 열고 아래 명령어를 딱 한 줄만 실행하시면 됩니다. 
데이터가 날아가지 않도록 볼륨(-v)까지 안전하게 매핑해 두었습니다.

# Bash
docker run -d \
  --name influxdb \
  -p 8086:8086 \
  -v influxdb2_data:/var/lib/influxdb2 \
  influxdb:2.7

# 설명 
-p 8086:8086: InfluxDB의 기본 포트입니다.

-v influxdb2_data:...: Docker Desktop을 껐다 켜도 센서 데이터가 초기화되지 않도록 디스크에 저장(Volume)합니다.

influxdb:2.7: 현재 가장 안정적인 2.x 최신 버전입니다.

초기 설정하기 (UI 접속)

컨테이너가 실행되었다면, 브라우저를 열고 http://localhost:8086 에 접속해 보세요. 파란색의 멋진 InfluxDB Web UI가 뜰 것입니다.

화면에 나오는 버튼을 눌러 딱 4가지만 입력해 주시면 세팅이 끝납니다.

Username: 관리자 아이디 (예: admin)

Password: 비밀번호 (예: admin1234)

Organization (Org): 조직 이름 (예: sensor_hq - 코드에서 써야 하니 기억하기 쉬운 걸로 적어주세요)

Bucket Name: 데이터를 담을 바구니 이름 (예: sensor_data)
